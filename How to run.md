# 🚀 How to Run — Linkedin-Zip-Solver

## Step 1: Clone the Repository

Open your terminal (Command Prompt / PowerShell / Git Bash) and run:

```bash
git clone https://github.com/Mohit-G-git/Linkedin-Zip-Solver.git
```

This will download the project folder to your computer.

---

## Step 2: Load as Unpacked Extension in Chrome

1. Open **Google Chrome**
2. Go to the Extensions page by typing in the address bar:
   ```
   chrome://extensions/
   ```
3. Enable **Developer Mode** by toggling the switch in the **top-right corner**
4. Click the **"Load unpacked"** button (appears after enabling Developer Mode)
5. In the file dialog, navigate to the cloned folder:
   ```
   Linkedin-Zip-Solver/
   ```
6. Select the folder and click **"Select Folder"**

---

## ✅ Done!

The **Linkedin-Zip-Solver** extension will now appear in your Chrome extensions list and be ready to use.

> 💡 **Tip:** Pin the extension to your toolbar by clicking the puzzle icon (🧩) in Chrome and pinning **Linkedin-Zip-Solver** for easy access.
